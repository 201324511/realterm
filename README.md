# 뱀게임
## 개발언어
  **C language**


## 게임방법

wasd키로 뱀을 움직여서 Food를 먹는다<br/>
많이 먹을수록 뱀은길어지고 속도도 빨라진다<br/>
자신의 몸에 머리가 닿거나, board의 경계에 머리가 닿게되면<br/>
게임은 끝이난다.<br/>

<pre><code>make</code></pre>
